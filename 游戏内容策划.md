# 霓虹幸存者 游戏内容策划

## 1. 游戏定位

《霓虹幸存者》是一个 Go + Ebitengine 制作的桌面飞机割草小游戏。核心体验是"雷霆战机 + 肉鸽三选一 + 数值膨胀"。玩家用鼠标控制战机移动，武器自动射击。击杀敌人获得经验和金币，经验用于局内升级，金币用于局外养成。

## 2. 基础操作

- 鼠标移动：控制战机位置。
- 自动射击：无需按键。
- 空格/鼠标右键：暂停/继续。
- 菜单数字键 1-4：购买局外养成。
- 彩蛋：菜单输入"上上下下左右左右BA"解锁无限金币。

## 3. 局内循环

1. 进入游戏，可能先选择初始抽卡（局外养成购买）。
2. 自动射击击杀敌人。
3. 经验球和金币按拾取范围吸附到玩家身上。
4. 经验满后触发三选一升级（带权重）。
5. 每波敌人逐渐增强，第 5 波后翻倍节奏。
6. 首领出现后进入更强数值膨胀阶段。
7. 击杀 3 个 BOSS 后敌人血量开始根据玩家输出调整。

## 4. 波次和难度曲线

- 第 1-4 波：基础成长阶段。
- 第 5 波开始：敌人属性进入翻倍膨胀（`enemyGrowthMultiplier`）。
- 第 7 波：小怪生命额外加速成长，变肉。
- 第 14 波前：怪物密度保持平缓，14 波后开始明显增密。
- 首个 BOSS 被击败后：刷怪节奏和敌人种类进入高压阶段。

核心函数位置：
- `update.go`：`waveGrowth()`、`enemyGrowthMultiplier()`、`spawnGrowthMultiplier()`、`expThreshold()`
- `entities.go`：`spawnEnemies()`、`updateWave()`、`spawnBoss()`

## 5. 玩家升级经验曲线

- 1-20 级：三次曲线增长（约 6→3746）。
- 21-49 级：指数增长 x1.12/级（约 3746→106k）。
- 50-100 级：陡峭指数 x1.22/级，封顶 80000。
- 怪物经验随波次倍数增长，后期刷怪不会卡级。

## 6. 玩家武器系统

### 基础武器

- 正前方自动子弹。
- 多重射击：增加前方子弹数量。堆叠多时子弹呈扇形散射（3 发以内窄扇面，4+ 发宽扇面扩展）。
- 扩散射击：开火角度变大。
- 侧翼炮：左右前方增加射击点。
- 尾翼炮：向后方射击。

### 质变武器（互斥，拿一次不再出现）

- 双重战机：左右小战机协同射击 + 视觉僚机。
- 前向激光：多角度前方攻击。
- 激光穿透：高伤害长射线，替换普通射击。
- 连锁闪电：命中后连锁附近最多 4 个敌人，带闪烁电弧特效。

### 互斥规则

`UpgradeTagExclusive` 类武器只会出现一个。Ultimate/Exclusive 标签卡牌拿过后不再重复出现。

## 7. 局内升级系统

升级为三选一，采用加权概率而非简单随机。

### 普通卡（可重复拿）

- 攻击强化：伤害 +25%
- 快速射击：射速 +20%
- 弹速提升：子弹速度 +25%
- 生命强化：最大生命 +25
- 紧急维修：恢复生命
- 推进器：移动速度提升
- 磁力装置：拾取范围 x3.2

### 稀有卡（可重复拿）

- 多重射击
- 扩散射击
- 侧翼炮
- 致命一击：暴击率 +12%
- 暴击伤害：暴击伤害 +60%
- 纳米修复：回血
- 巨型弹头：子弹伤害 +2
- 装甲镀层：最大生命 +40

### 金卡/质变卡（拿一次不再出现）

- 双重战机：左右小战机协同射击 + 视觉僚机。
- 激光穿透：高伤害长射线，替换普通射击。
- 连锁闪电：命中连锁最多 4 个敌人，电弧闪烁。
- 前向激光：多角度前方攻击。
- 护盾核心：周期免疫一次攻击（12s 冷却）。
- 歌莉娅巨人：体型+35%，获得 15% 自适应属性。
- 飞身踢：处决低生命敌人（16%血）并治疗。
- 巨人杀手：体型-25%，移速+20%，对大怪造成额外 10-30% 伤害。
- 灵魂虹吸：暴击率+25%，暴击治疗 12%。
- 渴血：获得 15% 全能吸血。
- 暴击律动：暴击可叠加攻速，最高 12 层。
- 会心防御：按暴击率触发减伤，最大 50% 概率减伤 55%。
- 终极标记：每 28s 标记 5s，记录 75% 伤害并在结束时引爆。
- 男爵之手：伤害+25%，弹速+15%，有双机时强化侧翼。

### 概率权重规则

- 1-2 级：金卡权重 10，紫卡 6，蓝卡 4（金卡占一半以上）。
- 3-5 级：金卡权重 6，紫卡 5，蓝卡 4。
- 6-8 级：金卡权重 3，紫卡 4，蓝卡 5。
- 9+ 级：金卡权重 1，紫卡 3，蓝卡 4（后期金卡稀有）。
- Ultimate/Exclusive 标签卡牌拿过后不再重复出现。
- Stat/Rate/Count 标签卡牌可重复出现。

核心文件：`upgrade.go`

## 8. 敌人系统

### 普通怪（type 0）

- 低血量基础敌机，从顶部生成。
- 后期血量随波次快速膨胀。

### 精英怪（type 1）

- 血量较高，斜向移动带边界反弹。
- 有血条。击败后有概率生成斩击波。
- 击杀 3 个 BOSS 后血量根据玩家输出调整。

### 射手怪（type 2）

- 发射追踪弹。8 波后有概率发射短寿命高伤射线。
- 击杀 3 个 BOSS 后血量根据玩家输出调整。

### 旋转怪（type 3）

- 发射 8 方向弹幕，产生压力。
- 有血条。击败后有概率生成斩击波。
- 击杀 3 个 BOSS 后血量根据玩家输出调整。

### 四面来敌（type 5）

- 波次 8+ 后出现，从屏幕四边冲入。
- 发射十字冲击弹，制造包围感。

### 首领（type 4）

- 每 3 波出现一次，击败后波次 +1、刷怪加速。
- 生命值由波数保底 + 击败 3 个 boss 后玩家输出估算取最高。
- 发射 12 方向旋转弹幕。击败后掉落 5 个道具 + 大量资源。
- 击败后有概率生成大型斩击波。

### 敌人属性对标规则

- 击杀 3 个 BOSS 前：敌人血量纯按波数。
- 击杀 3 个 BOSS 后：精英和 BOSS 血量取 `max(波数保底, 玩家输出×系数)`。
- 射速不参与敌人血量计算。
- 玩家输出估算含：伤害、子弹数、侧翼、双机、前弧、激光、闪电（x1.25）、暴击。

## 9. 特效与视觉

- 左上 HUD 显示 4 行属性：伤害/射速/暴击、子弹数量/散射/侧翼、移速/拾取/弹速、吸血/回血/护盾。
- 被动/能力标签不显示在 HUD 中。
- 金币和经验球：小而清晰，无光晕。
- 小怪死亡：轻量粒子。
- 大怪和首领死亡：明显爆炸。
- 激光光束：外层辉光+中层能量色+白色核心+端点亮斑。
- 连锁闪电：3 分支随机电弧 + 末端迸射火花。
- 斩击波：旋转双刃圈，带拖尾。

## 10. 资源吸附

- 初始拾取范围 120。
- 磁力装置 x1 后 ≈ 384，x2 后接近全屏。
- 吸附速度随接近加快。

## 11. 局外养成

金币菜单购买永久成长：

| 按键 | 项目 | 价格 | 上限 |
|------|------|------|------|
| 1 | 初始血量 +25 | 180 | 无上限 |
| 2 | 初始子弹 +1 | 320 | 初始 3 发 |
| 3 | 加速卡（开局长按第 5 波 + 大量属性） | 520 | 1 次 |
| 4 | 初始抽卡 +1 | 260×(当前+1) | 3 次 |

存档文件：`meta_progress.json`、`highscore.txt`

## 12. 文件结构

| 文件 | 职责 |
|------|------|
| `main.go` | 程序入口 |
| `types.go` | 核心类型、存档、工具函数 |
| `update.go` | 主循环、菜单、购买、彩蛋、玩家移动、经验曲线 |
| `entities.go` | 射击、生成敌人、敌人 AI、资源吸附、波次 |
| `combat.go` | 碰撞、伤害、死亡、连锁闪电、终极标记 |
| `upgrade.go` | 升级池、权重概率、选择逻辑 |
| `draw.go` | 主绘制、菜单、子弹、资源球、图元工具 |
| `draw_sprites.go` | 玩家、敌人、BOSS、斩击、死亡特效 |
| `draw_ui_elements.go` | 血条、道具图标 |
| `draw_screens.go` | HUD、暂停、升级、结算 |
| `font.go` | 中文字体加载 |

## 13. 后续扩展入口

- 加新敌人：`entities.go` + `draw_sprites.go`
- 加新卡牌：`upgrade.go`
- 加局外养成：`MetaProgress` + 菜单购买 + 菜单绘制
- 加新特效：优先控制粒子量，保持干净
